# Getting Help

## GitHub Issues

For bug reports and feature requests, open an issue on our GitHub repository. Include a minimal reproducer and the version of `openagentmesh` you're using.

## Discord

Join our Discord server for community discussion, questions, and sharing what you're building with OpenAgentMesh.

## Enterprise Support

For commercial support, consulting, or partnership inquiries, contact us at [openagentmesh@progresslab.it](mailto:openagentmesh@progresslab.it).

## Contributing

Contributions are welcome. Before submitting a pull request, open an issue to discuss the change. This keeps effort focused and avoids surprises during review.

For bug fixes, include a failing test that demonstrates the issue. For new features, start with the use case: describe what you're trying to build and why the current API doesn't cover it.
